# Life Savings Calculator

**Life Savings Calculator** is a custom WordPress plugin that allows users to calculate potential life savings based on predefined financial inputs.  
The plugin is designed to be lightweight, user-friendly, and easily extendable.

---

## Features
- Front-end calculator for life savings estimation
- Dynamic calculation using JavaScript
- Clean and simple user interface
- Easily embeddable via shortcode
- Modular and maintainable code structure

---

## Technologies
- PHP (WordPress)
- JavaScript
- CSS
- Git & GitHub

---

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Marko369/life-savings-cavculator.git
2. Copy the plugin folder to:
   /wp-content/plugins/
3. Activate Life Savings Calculator from the     WordPress admin panel

## Usage

After activation, insert the calculator into any page or post using a shortcode:

[life_savings_calculator]


The calculator will render automatically on the front end.
